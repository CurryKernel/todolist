## 1. 项目概述

### 1.1 项目定位

"源计划" 是一款面向女性用户的 Windows 桌面待办事项管理工具，以王源应援绿色为视觉主题，风格简约薄荷，涵盖办公、生活、运动、饮食等多种分类场景。

### 1.2 核心目标

- 提供完整 CRUD 待办事项管理
- 本地 JSON 文件存储，用户自主选择数据路径
- 王源主题中度融入（配色+头像+语录+动效）
- 简约大气、适合女生使用的视觉设计
- 全功能：分类、优先级、日期、搜索、统计、换肤

### 1.3 成功标准

- 用户首次启动可指定数据存储路径
- 所有数据本地存储，无需网络
- 6 大核心模块功能完整可用
- 3 套主题换肤可切换
- 应用打包为 Windows 安装程序 (.exe)

## 3. 数据模型


### 3.5 数据字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | UUID string | 是 | 唯一标识 |
| title | string(200) | 是 | 任务标题 |
| description | string(2000) | 否 | 详细描述 |
| category | string | 是 | 分类 ID |
| priority | enum | 是 | high/medium/low |
| dueDate | ISO datetime|null | 否 | 截止时间 |
| completed | boolean | 是 | 是否完成 |
| completedAt | ISO datetime|null | 否 | 完成时间 |
| createdAt | ISO datetime | 是 | 创建时间 |
| updatedAt | ISO datetime | 是 | 更新时间 |
| order | number | 是 | 排序序号 |
| tags | string[] | 否 | 自定义标签 |

## 4. 功能规格

### 4.1 待办管理 (CRUD)

#### 4.1.1 添加任务
- 触发：侧边栏 "+" 按钮 或 快捷键 `Ctrl+N`
- 流程：弹出快速添加栏（顶部）→ 输入标题 → 回车确认
- 可展开填写详细信息（描述、日期、优先级、标签）
- 默认分类：上一次使用的分类
- 新任务插入到列表顶部

#### 4.1.2 编辑任务
- 触发：点击任务卡片 → 右侧详情面板
- 可编辑：标题、描述、分类、优先级、截止日期、标签
- 自动保存：失焦或 1.5 秒防抖后自动保存

#### 4.1.3 删除任务
- 触发：详情面板删除按钮 或 右键菜单
- 确认：弹出柔和的确认气泡（非模态弹窗）
- 支持撤销：删除后 3 秒内可撤销

#### 4.1.4 完成/取消完成
- 触发：点击任务卡片左侧圆形复选框
- 动效：复选框填充绿色 + 文字划线 + 小音符飘出
- 已完成任务自动移至列表底部

#### 4.1.5 拖拽排序
- 原生 HTML5 Drag & Drop
- 拖拽手柄在任务卡片左侧
- 排序结果实时保存

### 4.2 分类系统

- 预设 4 个不可删除分类：办公、生活、运动、饮食
- 支持自定义分类：添加/编辑/删除
- 每个分类配置：名称、图标（emoji/SVG）、颜色
- 侧边栏分类列表，显示每个分类的未完成任务数
- 点击分类过滤列表

### 4.3 优先级 & 截止日期

- 三级优先级：高（红珊瑚色）、中（暖黄色）、低（灰绿色）
- 优先级显示为小色条在任务卡片左侧
- 截止日期选择器（原生日期输入美化）
- 逾期任务：高亮标题为珊瑚色 + 温和提醒标记
- 今日到期任务：列表顶部"今日待办"分组

### 4.4 搜索 & 筛选

- 顶部搜索栏：实时搜索（300ms 防抖），匹配标题+描述+标签
- 筛选器：分类下拉、状态切换（全部/进行中/已完成）、日期范围
- 排序：创建时间、截止日期、优先级、手动排序
- 搜索结果高亮匹配文字
- 搜索栏清空按钮

### 4.5 数据统计

- 侧边栏底部小卡片："今日完成 3/8"
- 点击进入统计面板：
  - **总览卡片** — 总任务数、完成率、今日完成、连续天数
  - **分类占比** — 环形图 (Chart.js)，各分类任务数量分布
  - **周趋势** — 折线图，每日完成数趋势
  - **优先级分布** — 横向条形图
- 统计数据实时更新

### 4.6 主题换肤

三套薄荷绿变体（CSS 变量切换）：

| 主题 | 主背景 | 侧边栏 | 强调色 | 适用场景 |
|------|--------|--------|--------|----------|
| 🌿 晨露 Morning | `#F5F9F6` | `#E8F3EC` | `#7BC8A4` | 白天办公 |
| ☀️ 午后 Afternoon | `#F0F7F2` | `#DCE8E0` | `#6AB892` | 全天通用 |
| 🌙 夜幕 Night | `#EAF2ED` | `#D2E2D8` | `#5AA882` | 夜间护眼 |

切换方式：设置面板按钮组，即时切换无闪烁

### 4.7 首次启动设置流程

1. **启动应用** → 显示启动页（1.5 秒渐变动画，随机王源语录）
2. **首次检测** → 无 settings.json → 进入设置引导
3. **路径选择** → "请选择数据存储位置" + 原生文件夹选择对话框
4. **初始化** → 创建默认分类 + 空 todos.json + settings.json
5. **欢迎页** → 薄荷渐变 + "源计划，陪你元气每一天" + 王源侧影线条
6. **进入主页** → 开始使用

## 7. 边界条件 & 异常处理

### 7.1 数据安全

- **自动保存**: 每次修改后自动写入文件（300ms 防抖）
- **自动备份**: 每次启动时备份前一日数据到 backups/
- **写入失败**: 显示非阻塞 toast 提示，每 30 秒重试
- **JSON 损坏**: 尝试从最新备份恢复，失败则显示恢复引导
- **路径不可访问**: 启动时检测，提示用户重新选择

### 7.2 性能边界

- 单文件上限 5000 条 todo（超出提示归档）
- 搜索防抖 300ms
- 列表虚拟滚动（超出 100 条时启用）
- 统计图表延迟渲染（切换统计面板时）

### 7.3 窗口管理

- 默认窗口 1200×800，最小 900×600
- 关闭到托盘（可配置）
- 记住窗口位置和大小
